# 👨🏻‍💻 Introduction

## Objective

Learn about JAMStack (Javascript, API and Markup).

## This repository has

A Newsletter with:

- OAuth with GitHub.
- HandlessCMS to create posts.
- Gateway payments for subscriptions.

## Project

This project was done through a module of the [Rocketseat](https://rocketseat.com.br) course.
